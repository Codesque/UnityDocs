# Unity Basics - Audio Source And UI Elements 

## How to play audio in Unity ? 
- You can play audio in unity by using `AudioSource` component.
- `AudioClip` subsection is the place where we can attach our .mp3 .wav files to the game object 
- `Play On Awake` option makes the audio play by the beginning of the game. 

## How to create an UI element ? 
- Right click on the Hierarchy Tab. Select anything from the UI part. That will automaticly create a Canvas Object which includes the UI element with a EventSystem GameObject. 

## What does this EventSystem gameobject do ? 
- EventSystem GameObject is responsible for detecting input on the UI elements.
